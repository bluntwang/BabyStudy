# BabyStudy - 幼儿园趣味学习应用

一款专为幼儿园小朋友设计的趣味学习应用，通过丰富多彩的图文游戏，帮助3-6岁儿童在快乐中学习成长。

## 功能介绍

### 小班（3-4岁）
- **海洋动物认知** - 认识各种海洋动物，了解它们的名称和叫声
- **中英文播放** - 趣味中英文单词学习
- **专注力训练** - 提升小朋友注意力的图文游戏

### 中班（4-5岁）
- **进阶图文识别** - 更复杂的图片识别游戏
- **情绪讲解** - 学习识别和理解不同情绪
- **分享教育** - 教导小朋友如何分享事物，体验分享的快乐
- **真话与假话** - 培养诚实品格，区分真话和假话
- **想象力培养** - 丰富想象力的趣味图文

### 大班（5-6岁）
- **数字算术** - 简单的数字认知和加减法练习
- **拼音学习** - 声母、韵母的认读学习
- **拼音闯关** - 趣味拼音拼读挑战游戏

## 界面特色

- 🎨 卡通风格设计，色彩鲜艳，吸引小朋友注意力
- 👆 触摸交互，实时反馈，鼓励小朋友主动探索
- 📱 支持手机端使用，随时随地学习
- 🏆 得分系统和奖励机制，增强学习动力

## 技术栈

- **框架**: Flutter
- **状态管理**: Riverpod
- **本地存储**: SQLite + SharedPreferences
- **音效**: Audioplayers
- **动画**: Lottie

## 运行项目

```bash
# 安装依赖
flutter pub get

# 运行应用
flutter run
```
