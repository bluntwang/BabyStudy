# BabyStudy 儿童教育App设计方案

## 1. 项目概述

**项目名称**：BabyStudy
**项目类型**：跨平台移动应用（iOS + Android）
**核心功能**：针对幼儿园三个年龄段儿童设计的趣味学习游戏，专注提升小朋友的专注力和逻辑思维
**目标用户**：3-6岁幼儿园小朋友

## 2. 技术架构

| 类别 | 技术选型 |
|------|----------|
| 框架 | Flutter |
| 语言 | Dart |
| 状态管理 | Riverpod |
| 本地存储 | SharedPreferences（配置）+ SQLite（游戏记录） |
| 音频播放 | audioplayers |
| 动画 | Flutter内置动画 + Lottie |

**项目结构**：
```
lib/
├── main.dart
├── core/              # 核心配置、主题、路由
├── models/            # 数据模型
├── providers/         # 状态管理
├── services/          # 存储服务、音频服务
├── screens/           # 页面
│   ├── home/          # 首页、年龄段选择
│   ├── game/          # 游戏页面
│   └── common/        # 公共页面
├── widgets/           # 公共组件
└── games/             # 各游戏模块
    ├── shape_color/   # 小班-形状颜色识别
    ├── matching/      # 小班-配对游戏
    ├── number/        # 中班-数字启蒙
    ├── language/      # 中班-语言表达
    ├── drawing/       # 大班-绘画创作
    └── pinyin/        # 大班-拼音学习
```

## 3. 功能模块

### 3.1 年龄段与游戏对照表

| 年龄段 | 年龄 | 游戏类别 | 具体游戏 |
|--------|------|----------|----------|
| **小班** | 3-4岁 | 图形认知类 | 形状识别、颜色识别、大小识别 |
| | | 逻辑思维类 | 配对游戏、简单分类、连连看 |
| **中班** | 4-5岁 | 数字启蒙类 | 数数、简单加减法（1+1=？、数水果） |
| | | 语言表达类 | 看图说话、词汇学习 |
| **大班** | 5-6岁 | 想象创作类 | 自由绘画、故事创编 |
| | | 拼音学习 | 拼音认知、拼音小闯关 |

### 3.2 核心功能

**首页模块**
- 展示卡通吉祥物（baby.png）
- 三个年龄段选择入口（小班/中班/大班）
- 已解锁关卡显示

**游戏模块**
- 图文识别游戏（图片+选项）
- 拖拽匹配游戏
- 限时挑战游戏
- 绘画创作工具

**奖励系统**
- 星星奖励：每通关一次获得星星
- 累计奖励：集齐一定数量星星解锁新关卡
- 音效反馈：答对（欢快音效+鼓励语音）、答错（柔和提示音+鼓励再来一次）

## 4. UI/UX设计

### 4.1 视觉风格
- **整体风格**：卡通可爱风格
- **边角**：圆润边角，无尖锐元素
- **色彩**：柔和的糖果色系，避免刺眼颜色
- **按钮**：大尺寸按钮（方便小朋友手指点击，最小60x60）
- **吉祥物**：baby.png卡通形象作为引导角色

### 4.2 配色方案
- 主色：柔和蓝色 #5B9BD5
- 次要色：温暖橙色 #FF9F43
- 强调色：清新绿色 #2ECC71
- 背景色：淡粉色 #FFF5F5
- 文字色：深灰色 #333333

### 4.3 交互设计
- 触摸反馈：点击有缩放动画
- 拖拽交互：流畅的拖拽效果
- 过渡动画：页面切换有趣味性动画
- 声效配合：关键操作有对应音效

## 5. 数据存储

### 5.1 本地存储内容
- 当前选中的年龄段
- 各年龄段已解锁关卡
- 游戏得分和通关记录
- 累计星星数量
- 用户偏好设置

### 5.2 存储方式
- **SharedPreferences**：存储简单配置（当前年龄段、设置）
- **SQLite**：存储游戏记录（每局得分、通关时间、星星数量）

## 6. 游戏详细设计

### 6.1 小班游戏

**形状识别**
- 显示一个图形，从选项中选择相同形状
- 图形：圆形、方形、三角形、星形、心形
- 答对获得1星

**颜色识别**
- 显示彩色物品，选择正确颜色
- 颜色：红、黄、蓝、绿、橙、紫
- 答对获得1星

**大小识别**
- 显示不同大小的物品，排序大小
- 答对获得1星

**配对游戏**
- 翻转卡片，找到相同的图片配对
- 锻炼记忆力和专注力
- 完成获得2星

**连连看**
- 将相同的图标用线连接
- 考验观察力
- 完成获得2星

### 6.2 中班游戏

**数数游戏**
- 显示水果/动物，数出数量
- 数量范围：1-10
- 答对获得1星

**简单加法**
- 展示 "2 + 3 = ?" 类型的题目
- 提供4个选项
- 答对获得1星

**看图说话**
- 显示一幅图，朗读相关词汇
- 培养语言表达能力
- 完成获得2星

**词汇学习**
- 图片+文字+发音的组合学习
- 类别：动物、食物、玩具等
- 答对获得1星

### 6.3 大班游戏

**自由绘画**
- 提供画布和彩色画笔
- 自由创作或按主题绘画
- 保存作品

**故事创编**
- 提供故事开头和角色
- 组合图片编故事
- 完成获得3星

**拼音认知**
- 显示汉字，显示拼音
- 声母、韵母、整体认读
- 答对获得1星

**拼音闯关**
- 拼音组合游戏
- 看图选正确拼音
- 通关获得3星

## 7. 音效资源

### 7.1 背景音乐
- 主界面轻快背景音乐
- 游戏进行中背景音乐
- 循环播放，音量可调节

### 7.2 音效
- 点击按钮音效
- 答对正确音效（欢快）
- 答错提示音效（柔和）
- 获得星星音效
- 通关庆祝音效
- 切换页面音效

### 7.3 语音
- 鼓励语音："你真棒！"、"加油！"、"再试一次"
- 正确答案："回答正确"、"太厉害了"
- 错误提示："没关系，再想想"
